네이버 사용자 톡톡이란 사용자가 친구추가없이, 앱 다운로드없이 바로 대화하는 웹 채팅 서비스입니다.

채팅서비스인만큼 사용자/파트너 말풍선과 전체적인 레이아웃의 마크업이 주된 업무였습니다.
실시간으로 사용자가 데이터를 입력하기 때문에 그에 따라 유동적으로 적용될 수 있는 UI를 마크업 하였습니다.
또한 코드의 재사용성과 웹접근성을 고려하여 마크업 하였습니다.
