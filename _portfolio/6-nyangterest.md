## 개발 환경

- **Database** : mysql(>=8.0) - **host**: nyangterest.cqn96a5xogpk.ap-northeast-2.rds.amazonaws.com **version**:1.0
- **Framewrok** : Node.js(>=10.13.0), Express / React
- **Language** : javaScript
- **상태관리** : MobX
