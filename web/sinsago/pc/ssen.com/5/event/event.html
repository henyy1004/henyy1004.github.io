쎈닷컴 강좌로 시험공부하고 야식 받는 방법

이벤트 기간 내 쎈닷컴 강좌를 1강 이상 수강하면 자동으로 이벤트에 참여됩니다.

  • ※ 이벤트 기간 내 쎈닷컴 유료&무료강좌 수강 시 자동으로 이벤트에 참여됩니다.
  • ※ 추첨 200명에게 기프티콘을 증정하고 이벤트 당첨 여부는 매주 금요일 공지사항에서 확인 가능합니다.
  • ※ 기프티콘은 쎈닷컴 회원정보의 휴대폰 번호로 발송되며 미기입/오기입 시 기프티콘이 재발송되지 않습니다.
  • ※ 본 이벤트는 당사의 사정에 의해 예고 없이 변경 또는 종료될 수 있습니다.
